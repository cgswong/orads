Oracle Internet Directory
=========================

The following supplemental or dependent software is required for the successful setup of
the Oracle Internet Directory (OID) software stack:

  - Oracle Grid Infrastructure (GI) 11.2.0.4
    - p13390677_112040_Linux-x86-64_1of7.zip
    - p13390677_112040_Linux-x86-64_2of7.zip
    
  - Oracle Database 11.2.0.4
    - p13390677_112040_Linux-x86-64_3of7.zip
    
  - Oracle JDK 1.7u51
    - jdk-7u51-linux-x64.tar.gz
    
  - Oracle WebLogic Server (WLS) 10.3.6
    - wls1036_generic.jar

  - Repository Configuration Utility (RCU) 11.1.1.7
    - fmw_rcu-11.1.1.7_linx64.zip
